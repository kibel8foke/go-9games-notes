# go-9games-notes

本仓库用于归档和发布多个独立的 HTML 页面，作为各类笔记、演示页、工具页或临时内容的静态存储空间。所有页面均为纯前端实现，不依赖特定后端服务。

## 项目简介

go-9games-notes 是一个以 HTML 文件为主要载体的归档仓库，适合存放需要以网页形式呈现的零散内容，例如个人笔记、小工具原型、学习演示页面等。每个 HTML 文件独立运行，便于直接通过 GitHub Pages 或其他静态托管服务访问。

## 目录说明

仓库根目录下存放所有 HTML 页面文件，建议按功能或主题进行命名，以便快速识别。例如：

- `note-xxx.html`
- `tool-yyy.html`
- `demo-zzz.html`

目前不强制要求子目录结构，所有页面均位于根目录，便于直接链接和访问。

## 页面归档说明

每个 HTML 页面应自包含其所需的样式和脚本，避免引用外部不可控资源。页面内容可以是：

- 学习笔记与总结
- 小型交互工具或计算器
- 演示或原型页面
- 临时记录或实验性内容

不限制页面主题或用途，但请确保内容不违反平台使用条款。

## 维护说明

- 欢迎提交 Pull Request 添加新的 HTML 页面，或对现有页面进行修正。
- 页面命名应清晰、无歧义，避免使用特殊字符或空格。
- 建议在页面内添加作者或备注信息，方便后续维护。
- 本仓库不提供实时预览或动态更新功能，所有页面均为静态内容。
